## Getting Started

This project require NodeJS v12

### Install
Clone this project

```bash
$ git clone https://github.com/AkiRaID/akirabotv1.git
$ cd akirabotv1
```

Install the dependencies:

```bash
$ pkg install nodejs
$ pkg install ffmpeg
$ pkg install imagemagick
$ bash install.sh
```

### Usage
Run the Whatsapp bot

```bash
$ node index.js
```
